# Open-Lab.04.14

Vytvorte funkciu, ktorá vezme pole reťazcov a vypíše ich do konzoly, každý do jedného riadku, spolu v ráme.
	Napríklad pre list ["Hello", "World", "in", "a", "frame"] dostaneme výpis:
````
	*********  
	* Hello *  
	* World *  
	* in    *  
	* a     *  
	* frame *  
	*********  
````
